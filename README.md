# علي · ALI.DESIGN

البورتفوليو الشخصي لعلي، مصمم مواقع للأعمال السعودية.

- أكثر من 75 موقعًا منشورًا لكوفيات ومطاعم وسبا وعيادات ومكاتب هندسية في 7 مدن
- معاينات حية لمواقع حقيقية منشورة، مع فلترة حسب القطاع
- تصميم فخم داكن (أسود + ذهبي رملي)، عربي RTL بالكامل، متجاوب مع الجوال
- تواصل مباشر عبر واتساب، ومتابعة الأعمال الجديدة على تيك توك [ali.design02](https://www.tiktok.com/@ali.design02)
