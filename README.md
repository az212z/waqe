# وَقْع WAQE

استوديو سعودي يصمم وينشر مواقع للكوفيات والمطاعم والسبا والعيادات والمكاتب.

**موقع له وَقْع.**

- أكثر من 75 موقعًا منشورًا في 7 مدن سعودية، لـ 8 قطاعات مختلفة
- معاينات حية لمواقع حقيقية: معاينة ديسكتوب عريضة + رف سحب أفقي فيه 11 موقعًا
- تصميم عصري جريء: ورقي + حبر + أزرق كوبالت، خط Zain، عربي RTL بالكامل
- تواصل عبر واتساب، والشغل الجديد على تيك توك [ali.design02](https://www.tiktok.com/@ali.design02)
